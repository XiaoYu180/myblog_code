allure 和 pytest 相关环境安装
----------------------

```bash
# allure
pip3 install allure-pytest -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
#pytest
pip3 install pytest -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
```
